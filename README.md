## Hadoop과 Pig에 대한 소개

### Hadoop

Hadoop은 대량의 데이터를 분산 처리하기 위한 오픈 소스 프레임워크입니다. Hadoop은 여러 노드에서 데이터를 저장하고 처리할 수 있으며, MapReduce 프로그래밍 모델을 통해 병렬 처리를 수행합니다.

### Pig

Pig는 Hadoop에서 데이터 처리를 위한 스크립트 언어로, Pig Latin이라는 언어를 사용하여 데이터 흐름을 정의하고 처리할 수 있습니다. Pig는 복잡한 데이터 처리 작업을 더 간단하게 추상화하여 수행할 수 있게 해줍니다.

이 프로젝트는 Hadoop과 Pig에 대한 소개와 간단한 예시 코드, 그리고 이를 활용한 데이터 처리 방법을 다루고 있습니다.



<h3>목적</h3>
<p>1.직접 하둡과 피그를 통해 데이터를 다루어 사용법 익히기</p>
<p>2.빅데이터 처리 작업에 필요한 도구 학습하기</p>
<p>3.CLI 환경 학습하기</p>
<hr/>

<h3>하둡 & 피그의 이점</h3>
<p>1.대용량 데이터 처리: 하둡은 수백 테라바이트에서 페타바이트 규모의 대용량 데이터를 처리할 수 있습니다</p>
<p>2생태계 (Ecosystem): 하둡은 다양한 생태계 도구를 지원하며, 데이터 수집, 저장, 처리, 분석, 시각화, 감지 등에 사용할 수 있는 다양한 라이브러리와 프로젝트를 포함하고 있습니다.</p>
<p>3.피그는 복잡한 맵리듀스 작업을 쉽게 표현할 수 있는 고수준의 데이터 흐름 언어입니다. Java 기반의 맵리듀스 코드보다 훨씬 간단하게 복잡한 데이터 분석 작업을 처리할 수 있습니다.</p>
<br/>

<hr/>
<h3>요약</h3>
요약하면, 하둡은 대용량 데이터 처리, 분산 처리, 내결함성과 같은 기능을 제공하며, 피그는 복잡한 데이터 분석 작업을 간단하고 효율적으로 처리하기 위한 언어와 실행 계획 최적화를 제공합니다. 이 두 가지 프레임워크는 빅데이터 처리와 분석을 위해 필수적인 도구로 사용되고 있습니다.
